# GitHub Pages quick setup

This repository is ready to deploy via GitHub Actions.

## If you want `https://priyapyada.github.io`

1. Rename the repository to exactly `priyapyada.github.io`.
2. Keep the default branch as `main`.
3. Go to **Settings -> Pages**.
4. Under **Build and deployment**, set **Source** to **GitHub Actions**.
5. Push to `main` and wait for **Deploy GitHub Pages** to turn green.

Your site will be live at:
`https://priyapyada.github.io`

## If you keep any other repository name

The URL will be:
`https://priyapyada.github.io/<repository-name>/`

## Site paths

- Main page: `/` (from `index.html`)
- Alternate page: `/v2/`
